Prework Study Guide

HTML

  • < !DOCTYPE > - Declares to the browser that the document is in HTML
  • < html lang="" > - Declares to the browser the language used
  • < head > - The container for metadata (< meta >)
  • < meta > - contain information about the page that is used by the browser and won't be seen by the user
  • charset="" - Sets the character encoding
  • http-equiv - Sets an HTTP header for the content
  • viewport - Sets the default view of the webpage
  • < title > - Titles the webpage tab
  • < bodty > - Defines the body content that will be seen by the user
  • < div > - Creates a division in the page
  • < span > - Creates an inline division in the page
  • < h1 > - A level of heading (h1-h6)
  • < p > - A paragraph block of text
  • < ul > - Unordered list of content
  • < ol > - Ordered list of content
  • < li > - Items of content within a list
  • < img > - Used to contain various attributes for an image
  • scr - Defines the location of the image
  • alt - Text string that describes the image
  • width & height - Sets the size of the image
  • < a > - An anchor that creates a link to the same or other webpages
  • href - attributes points to the URL for the link
  • < !-- - Comments that are used in the code that are not visable to users
  • < footer > - Appears at the bottom of a page
  • < section > - Semantic elements that define a section in a document
  • class="" - Assigns a class to the specific content
  • id - Attribute that identifies elements

CSS

  • < style > - Changes the color of every text within it. Used in the (< head >)
  • color: - Used to change the color of the text. Color can be refrenced by hexadecimal code or hex #
  • < link > - Used to refrence an external style sheet
  • rel - Specifies the relationship between the current document and the linked document resource
  • href - Details the location from the current location
  • margin - Indicates how much space around the outside of an element
  • padding - Indicates how much space within the inside of an element
  • box-shadow - adds a shadow effect around an element's frame
  • * - Declares all elements on a page to match the same style properties within it
  • text-align - Sets the text to a certain side of the page
  • Selector - Defines the elemt or attribute to which declarations will apple
  • Declarations - Contains a CSS property that we apply a value to

Git

  • ls - Lists the directory in the file chosen
  • cd - Selects a file in the current directory
  • cd .. - Goes back one stage in the file directory
  • git - Command that must precede EVERY Git statement
  • checkout - Command that is used to move to different branches
  • -b - Command to create a new branch
  • add - Adds modifications to the current working branch
  • -A - Command to commit all current changes in the working branch
  • -m - Associate a message to the commit
  • pull - Command to receive brance modifications into the local enviroment. Always good to use to check if you are on the latest version on your project
  • origin - Keyword that indicates the source of the pull will be in the GitHub repo
  • main - Indicates the branch
  • push - Moves the changes made locally to our remote GitHub branch.
  • touch - Creates a file
